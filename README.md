##  Скопировать объект при помощи рекурсивной функции.

При помощи рекурсивной функции необходимо скопировать данный объект:

user = {
    name:'Bob',
    age: 34,
    id:1,
    address:{
        city:'Odessa',
        country:'Ukraine'
    },
    books:[
        {
            name:'red one',
            id:35,
            author: {
                name:'Some Author',
                id:277
            },
            tags:['history', 'roman']
        }
    ]
